This is a side-project developed to test and practice js and css skills. The goal was to make a simple tic tac toe game for 2 players or 1 player vs AI.

## Roadmap

- Player win counter
- Difficulty modes for the AI. Easy which is the current one and medium which is almost the same but will choose a corner if starting and the center if not on the first move. Difficult will be a minimax algorithm based AI.
- Modify the gameover screen so that you can check back on the board.
